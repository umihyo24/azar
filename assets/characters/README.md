# characters assets

このフォルダにはキャラクター画像をそのまま入れてください。

## 置き場所

```text
assets/characters/
```

## 画像ファイル名

```text
normal_normal.png
normal_charge.png
speed_normal.png
speed_charge.png
sanma_normal.png
sanma_charge.png
```

## 補足
- 画像が無い場合でもゲームは落ちません。
- 画像が見つからないときは、Canvas 上に簡易プレースホルダーが表示されます。
- 前回入っていた `.gitkeep` は「空フォルダを Git に残すためのダミーファイル」でした。
  画像を置くときに気にしなくて大丈夫です。
